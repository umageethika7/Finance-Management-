# Finance Project

## 📌 Project Description

This is a full stack web application developed to manage financial data.
It includes a backend for handling APIs and a frontend for user interaction.

---

## 🛠️ Technologies Used

* Java (Backend)
* Spring Boot
* MySQL
* HTML, CSS, JavaScript (Frontend)
* React (if you used)

---

## ⚙️ Setup Steps

### 🔹 Backend Setup

1. Open backend project in Eclipse 
2. Configure MySQL database
3. Run the Spring Boot application
4. Server will start on: http://localhost:2527

### 🔹 Frontend Setup

1. Open frontend folder in VS Code
2. Install dependencies:
   npm install
3. Run the project:
   npm start
4. Application will open in browser
