## 📌 Project Description

# Finance Management System

The Finance Management System is a full-stack web application designed to help users manage their daily financial transactions efficiently. This application allows users to record income and expenses, view financial data in a structured table format, and track their overall balance.
The frontend is developed using React (Vite) and Mantine UI, providing a clean and responsive user interface. The backend is built using Spring Boot, which handles all CRUD operations and API requests.
Users can add new financial records, edit existing entries using a pre-filled form, and delete records with confirmation. The application also supports sorting data based on different fields such as date, amount, and category.
Additionally, the system automatically calculates total income, total expenses, and the remaining balance, displaying them in a clear and visually distinct format.
This project demonstrates the integration of frontend and backend technologies, REST API communication using Axios, and effective state management in React.

## ⚙️ Setup Steps

### 🔹 Backend Setup

1. Open backend project in Eclipse 
2. Configure MySQL database
3. Run the Spring Boot application
4. Server will start on: http://localhost:2507

### 🔹 Frontend Setup

1. Open frontend folder in VS Code
2. Install dependencies:
   npm install
3. Run the project:
   npm start
4. Application will open in browser
